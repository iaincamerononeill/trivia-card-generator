Privacy Policy

Last updated: February 11, 2026

Overview

This Trivia Card Generator is designed with privacy in mind. We do not collect, store, or share any personal information.

What We Don't Collect

  • No account required - No registration or login needed
  • No personal data - We don't ask for names, emails, or contact details
  • No tracking - No cookies, analytics, or tracking pixels
  • No file storage - Uploaded CSV files are immediately deleted after processing

How Your Data is Handled

CSV Upload Method

  • Files are uploaded to temporary server storage
  • PDF is generated from your CSV data
  • Both CSV and PDF are automatically deleted immediately after download
  • No data is retained on our servers

AI Generation Method

  • Your topic, level, and API key are sent to the selected AI provider
  • Generated questions are processed into a PDF
  • No data is stored after generation
  • API Keys: Your API key is transmitted directly to the AI provider and never stored by us
  • Please review your AI provider's privacy policy:
    • OpenAI Privacy Policy
    • Anthropic Privacy Policy
    • Google Privacy Policy

Security

  • All connections use HTTPS encryption
  • File uploads are limited to 10MB
  • Only CSV files are accepted
  • Files are processed in isolated temporary directories
  • API keys are never logged or stored
  • Security headers are set to prevent common web vulnerabilities

Third-Party Services

When using AI generation, your request is processed by the AI provider you select:

  • OpenAI (if selected) - Subject to OpenAI's privacy policy and terms of service
  • Anthropic (if selected) - Subject to Anthropic's privacy policy and terms of service
  • Google (if selected) - Subject to Google's privacy policy and terms of service

We have no control over and assume no responsibility for the content, privacy policies, or practices of any third-party sites or services.

Your Rights

Since we don't collect or store any personal data, there is no data to access, modify, or delete. Each session is completely isolated and temporary.

Educational Use

This tool is designed for educational purposes. Users are responsible for:

  • Ensuring their question content complies with applicable laws
  • Respecting intellectual property rights
  • Not using AI-generated content in ways that violate their AI provider's terms of service
  • Reviewing AI-generated content for accuracy before use

Children's Privacy

This service does not knowingly collect personal information from children. The tool is intended for educational content creation by adults (teachers, parents, educators).

Data Retention

Zero retention policy:

  • Uploaded files: Deleted within seconds after PDF generation
  • Generated PDFs: Deleted after download
  • API keys: Never stored, only used for immediate API calls
  • Server logs: Standard web server logs are rotated regularly and contain no personal information

Changes to Privacy Policy

Any changes to this policy will be reflected on this page with an updated date. Continued use of the service after changes constitutes acceptance of the updated policy.

Contact

This is an independent educational project. For questions about privacy, please review the source code on our GitHub repository (if available) or discontinue use if you have concerns.

Legal Basis for Processing

We process data based on:

  • Legitimate interests - Providing the card generation service
  • User consent - By using the service, you consent to temporary file processing
